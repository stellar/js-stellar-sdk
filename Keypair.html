Keypair

Keypair

new Keypair(keys)

Keypair represents public (and secret) keys of the account.

Use more convenient methods to create Keypair object:

  • Keypair.fromAccountId
  • Keypair.fromSeed
  • Keypair.random
Parameters:
Name Type Description
keys object
Properties

Raw public key
