Class: Transaction

Transaction

new Transaction(envelope)

A new Transaction object is created from a transaction envelope (or via TransactionBuilder). One a Transaction has been created from an envelope, its attributes and operations should not be changed. You should only add signers to a Transaction object before submitting to the network or forwarding on to additional signers.
Parameters:
Name Type Description
envelope string | xdr.TransactionEnvelope The transaction envelope object or hex encoded string.
