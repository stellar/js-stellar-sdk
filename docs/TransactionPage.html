Class: TransactionPage

TransactionPage

new TransactionPage(json)

A TransactionPage contains the results of a transaction collection query, such as transactions or account_transactions. It holds a list of transaction results, which are transaction json objects. The TransactionPage can be passed as a parameter to the Server.nextTransactionPage() or Server.previousTransactionPage() to return the next/previous page of transactions for this query.
Parameters:
Name Type Description
json object The response JSON from Horizon.
